# Privacy Shadow Web (Archived)

**Status**: ARCHIVED - Educational web app built for FOSSASIA Hackathon 2026

This was the original Next.js educational web application that demonstrated digital footprint concepts through a "Digital Twin" visualizer.

## What Was Built

- Interactive Digital Twin avatar that grew based on data exposure
- EXIF metadata analyzer for uploaded photos
- Form filler simulator showing data capture
- Social risk simulator demonstrating friend exposure
- Privacy recommendations and action items

## Archiving Reason

Pivoting from educational web app to active browser extension with real-time protection:
- Real-time PII (Personal Identifiable Information) detection
- ML-powered stranger detection
- Parent notification system
- Active intervention before sharing

## Migration Notes

### Reusable Components
- Icons and assets from `src/components/`
- Styling patterns from `src/app/globals.css`
- Copy and messaging from original README

### What's Being Rebuilt

- Backend API (new alert delivery)
