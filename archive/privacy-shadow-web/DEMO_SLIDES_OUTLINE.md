## 🎯 Presentation Tips

### **Timing**
- Total: 10-12 minutes
- Slides: 5-7 minutes
- Demo: 5 minutes
- Q&A: 3-5 minutes

### **Key Messages**
1. Make the invisible VISIBLE
2. Empower, don't restrict
3. Privacy by design
4. Real impact, real validation

### **Demo Flow**
1. Start with empty twin
2. Add data (photo)
3. The reveal (twin grows)
4. The solution (twin diet)
5. The result (twin shrinks)

### **Storytelling**
- Problem → Solution → Impact
- Use emotional language ("Whoa!", "Forever")
- Show, don't just tell
- End with call to action
