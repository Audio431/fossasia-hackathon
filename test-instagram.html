📱 Instagram Integration Test

Verify Privacy Shadow works on Instagram.com

✅ Pre-Test Checklist

1 Extension Loaded
Open Chrome and go to chrome://extensions/
Verify "Privacy Shadow" is listed and enabled
2 Console Open
Open Instagram.com (you'll need to be logged in)
Press F12 to open Developer Tools
Go to the Console tab
3 Check for Logs
You should see these messages in the console:
Privacy Shadow: Instagram form monitor active Privacy Shadow: Form monitor active Privacy Shadow: Initializing Instagram form monitor Privacy Shadow: Monitoring Instagram DMs Privacy Shadow: Found X potential input elements

🧪 How to Test on Instagram

1 Navigate to DMs
Go to instagram.com/direct/inbox/
Or click the messenger icon in the top right
2 Open a Conversation
Click on any DM conversation
You should see the message input field at the bottom
3 Type Test Message
In the message input, type:
my address is 123 main street springfield
4 Trigger Detection
Click outside the message box (blur the input)
Or press Enter to try to send
5 Verify Alerts
You should see:
  • 🎯 Alert overlay warning you about the address
  • 📱 Green WhatsApp notification slides in from the right
  • 💬 Console shows "PII detected in Instagram DM"

⚠️ Common Issues & Solutions

Issue 1: No console logs appear
Solution: Make sure you rebuilt the extension after adding the Instagram monitor:

npm run build
Then reload the extension in chrome://extensions/

Issue 2: Console shows "Found 0 potential input elements"
Solution: Make sure you're actually on a DM conversation page, not just the inbox. Try:

instagram.com/direct/t/1234567890/
(any conversation ID will work)

Issue 3: Alert appears but WhatsApp notification doesn't slide in
Solution: Check the console for errors. The notification simulator might have failed. Make sure the notification-simulator.ts file exists in the build output.

Issue 4: Instagram doesn't load in Chrome
Solution: Some Instagram features require being logged in. Make sure you're logged into your Instagram account in Chrome.
