   - Point out extension icon

2. **Safe Conversation** (30s)
   - Select "Safe Conversation"
   - Show: 15 mutual friends, 45 interactions
   - Result: 8% risk (green)

3. **Stranger Danger** (30s)
   - Select "Stranger Danger"
   - Show: 0 mutual friends, new account
   - Result: 78% risk (orange)

4. **Grooming Pattern** (30s)
